# Homework-Strategist
LaserHacks project made by Team 2, intended to provide user optimal planning for their courses.

# Description
Allows the user to have their Canvas course work dates be planned out on their Google Calendar.

- Java version 17 Minimum (25/26 Recommended)
- Python Version +3.10
